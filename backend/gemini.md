This file provides instructions and context for Gemini to understand and interact with the backend of this project.

## Backend Overview

The backend is a Node.js application written in TypeScript. It uses Express.js for the server and routing.

**Key Technologies:**
- Node.js
- TypeScript
- Express.js

**Project Structure:**

- `src/`: Contains the main source code.
- `src/server.ts`: The entry point of the application.
- `src/routes/`: Defines the API endpoints.
- `src/services/`: Contains the business logic, such as database interactions and web scraping.
- `src/models/`: Defines data structures and types.
- `src/middleware/`: Contains middleware for things like authentication.
- `src/utils/`: Contains utility functions.
