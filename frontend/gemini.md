This file provides instructions and context for Gemini to understand and interact with the frontend of this project.

## Frontend Overview

The frontend is a React application written in TypeScript and built with Vite.

**Key Technologies:**
- React
- TypeScript
- Vite
- Material-UI (or other component library if used)
- i18next for internationalization

**Project Structure:**

- `src/`: Contains the main source code.
- `src/main.tsx`: The entry point of the application.
- `src/App.tsx`: The main App component.
- `src/components/`: Contains reusable React components.
- `src/services/`: Contains code for interacting with the backend API.
- `src/contexts/`: Contains React contexts for state management.
- `src/i18n/`: Contains internationalization configuration and locale files.
- `src/utils/`: Contains utility functions.
